# Week 3 - Playwright UI Automation

First hands-on browser automation with Playwright, building up to the Page Object Model.

## Contents
- `day11_first_script.py` / `day11_first_test.py` - first Playwright script and test
- `day12_locators.py` - locating elements
- `day13_assertions.py` - assertions with expect
- `day14_pom_test.py` - first Page Object Model test
- `day15_multi_page_test.py` - a flow across multiple pages
- `pages/` - Page Objects for the DemoQA checkbox, radio button, and text box widgets
- `*.png` - reference screenshots captured during the exercises

## Target
Tests run against demoqa.com.

## Running
```
python -m pytest day14_pom_test.py -v
```
