# Week 7 - SQL Fundamentals

SQL for QA data validation, using a local SQLite database.

## Contents
- `setup_db.py` - builds the SQLite school database
- `day1_queries.py` through `day5_queries.py` - SELECT/WHERE/ORDER BY, aggregates, JOINs, subqueries, and NULL handling
- `school.db` - the generated database

## Running
First build the database, then run any day's queries:
```
python setup_db.py
python day1_queries.py
```
